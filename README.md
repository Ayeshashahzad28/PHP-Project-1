# 🎓 PHP Student Registration Form

A simple **Student Registration System** built using **PHP and MySQL**. This project allows users to submit student details through a form, and the data is stored in a MySQL database. An admin panel is provided to view and manage all registered students.

---

## 🚀 Features

- Student registration form  
- Data storage in MySQL database  
- Admin panel to view student records  
- Clean and simple UI  
- Basic CRUD structure (expandable)  
- Responsive layout (optional improvement)  

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- PHP  
- MySQL  
- XAMPP / Local Server
